\f0\fs24 \cf0 PROJECT TITLE: DiabeTrack \'96 Smart Diabetes Monitoring and Lifestyle Assistant\
\
DESCRIPTION:\
DiabeTrack is a smart diabetes management system designed to help patients monitor glucose levels, receive medication reminders, and gain lifestyle insights through data visualization and prediction.\
\
---------------------------------------\
STEPS TO EXECUTE THE PROJECT\
---------------------------------------\
\
1. Clone or download the repository from GitHub:\
   https://github.com/amirthapec117-crypto/DiabeTrack_MiniProject_Team-A12\
\
2. Extract the project folder if downloaded as a ZIP.\
\
3. Open the "Source_Code" folder.\
\
4. Run the FRONTEND (React App):\
   - Navigate to Source_Code/frontend\
   - Open terminal and run the following commands:\
        npm install\
        npm start\
   - This will open the app at http://localhost:3000\
\
5. Run the BACKEND (Flask Server):\
   - Navigate to Source_Code/backend\
   - Install dependencies using:\
        pip install -r requirements.txt\
   - Start the Flask server using:\
        python app.py\
   - The backend will start at http://localhost:5000\
\
6. Connect Bluetooth glucometer via the web app.\
\
7. Login using Firebase credentials and start tracking glucose readings.\
\
---------------------------------------\
TECHNOLOGIES USED\
---------------------------------------\
Frontend  : React  \
Backend   : Flask  \
Database  : Firebase  \
APIs      : Google Fit API, Bluetooth Glucometer  \
Libraries : Chart.js for Visualization  \
ML Model  : Risk Prediction for Nocturnal Hypoglycemia  \
